# 🚀 Facebook / Reel Video Downloader PRO

Modern arayüzlü, çoklu link destekli, otomatik etiket çıkaran güçlü bir video indirme aracı.

---

## ✨ Özellikler

* 🎬 Facebook Reel & Video indirme
* 🔗 Toplu link desteği (birden fazla URL)
* 📊 Canlı progress bar (% gösterimi)
* 🌙 Modern Dark UI (Metro style)
* 📁 Otomatik `downloads` klasörü oluşturma
* 🏷️ Hashtag (#etiket) çıkarma
* 🧠 WordPress uyumlu içerik desteği
* ⚡ Hızlı ve donmadan çalışan sistem (threading)

---

## 🖼️ Arayüz

* Linkleri toplu şekilde gir
* İndir butonuna bas
* Videolar otomatik indirilsin
* Etiketler altta listelensin

---

## 📥 Kurulum

### 1. Python kurulu olmalı

👉 Python 3.8+

### 2. Gerekli kütüphaneleri yükle

```bash
pip install yt-dlp
```

---

## ▶️ Çalıştırma

```bash
python app.py
```

---

## 🧪 Kullanım

1. Programı aç
2. Video linklerini alt alta yapıştır:

```
https://www.facebook.com/reel/xxxx
https://www.facebook.com/reel/yyyy
```

3. **İNDİR** butonuna bas
4. Videolar otomatik olarak:

```
/downloads
```

klasörüne kaydedilir

---

## 🏷️ Hashtag Sistemi

Program video başlığından otomatik olarak:

```
#haber #trafik #istanbul
```

etiketlerini çeker ve listeler.

👉 Bunları direkt:

* WordPress tag
* SEO keyword
* içerik açıklaması

olarak kullanabilirsin.
